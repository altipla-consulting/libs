
# connect

[![GoDoc](https://godoc.org/libs.altipla.consulting/connect?status.svg)](https://godoc.org/libs.altipla.consulting/connect)

Package `connect` has GRPC connections utils.


### Install

```go
import (
	"libs.altipla.consulting/connect"
)
```
